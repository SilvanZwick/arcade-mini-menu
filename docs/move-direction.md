# move direction

A movement direction. This block is meant to be used with the `move selection` block.

```sig
miniMenu._moveDirectionn(miniMenu.MoveDirection.Up)
```

The different move directions each correspond to a number:

* **up** = 0
* **down** = 1
* **left** = 2
* **right** = 3
